## 📊 技術架構

### 共用基礎設施

所有 Agent 共用以下基礎設施：

```
backend/app/
├── agents/                    # Agent 核心
│   ├── base_agent.py         # 基礎 Agent 類別
│   ├── orchestrator.py       # Agent 協調器
│   └── memory/               # 記憶系統
│       ├── short_term.py     # 短期記憶 (Redis)
│       └── long_term.py      # 長期記憶 (PostgreSQL)
├── services/
│   ├── rag_service.py        # RAG 檢索增強生成
│   ├── vector_service.py     # 向量搜尋
│   └── llm_service.py        # LLM 整合 (已存在)
└── schemas/
    └── agent_schemas.py      # Agent 相關 Schema
```

### 技術棧

- **LLM**: Groq (Llama 3.3 70B)
- **向量資料庫**: pgvector (已有)
- **快取**: Redis (新增)
- **Agent 框架**: LangChain (新增)
- **記憶管理**: 自定義 + Redis
