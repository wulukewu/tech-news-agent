### й–Ӣзҷјз’°еўғзү№й»һ

1. **Volume жҺӣијү**
   - Backend: `./backend/app` вҶ” `/app/app` (йӣҷеҗ‘еҗҢжӯҘ)
   - Frontend: `./frontend` вҶ” `/app` (йӣҷеҗ‘еҗҢжӯҘ)
   - Logs: `./backend/logs` вҶ” `/app/logs` (жҢҒд№…еҢ–)

2. **Hot Reloading**
   - дҝ®ж”№жң¬ең°жӘ”жЎҲ вҶ’ иҮӘеӢ•еҗҢжӯҘеҲ°е®№еҷЁ вҶ’ иҮӘеӢ•йҮҚж–°ијүе…Ҙ

3. **й–Ӣзҷје·Ҙе…·**
   - еҢ…еҗ«жүҖжңүй–Ӣзҷјдҫқиіҙ
   - е•ҹз”ЁйҷӨйҢҜжЁЎејҸ
   - и©ізҙ°зҡ„йҢҜиӘӨиЁҠжҒҜ
